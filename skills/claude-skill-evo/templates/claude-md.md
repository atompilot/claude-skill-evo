## Skill 自我进化

本项目的 skill 体系具备被动进化能力。在日常工作中，如果你发现：
- **skill 内容与项目现状不符**（路径、命令、API 已变）→ 提议更新 skill
- **用户纠正了 skill 中的规范** → 提议将正确做法写入 skill
- **反复出现的操作模式未被 skill 记录** → 提议补充
- **用户明确说"记住"、"以后都这样"** → 提议写入 skill

修改 skill 前必须向用户确认，绝不擅自修改。
